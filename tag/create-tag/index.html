标签总是和某个 commit 挂钩。如果这个 commit 即出现在 master 分支，又出现在 dev 分支，那么在这两个分支上都可以看到这个标签。

小结

  • 命令git tag <tagname>用于新建一个标签，默认为 HEAD，也可以指定一个 commit id；
  • 命令git tag -a <tagname> -m "blablabal..."可以指定标签信息；
  • 命令git tag可以查看所有标签。
